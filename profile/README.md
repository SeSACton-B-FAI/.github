# 비파이 (B-FAI)

어르신과 교통약자를 위한 대중교통 안내견 서비스

---

## 왜 만들었나요

할머니께서 지하철 타실 때마다 고생하시는 걸 봤습니다. 어느 출구에 엘리베이터가 있는지, 몇 번째 칸에 타야 내릴 때 편한지, 매번 물어보시고 헤매십니다.

비파이는 그런 분들을 위한 안내견입니다. 지금 서 계신 위치에서 가장 가까운 엘리베이터 출구를 찾아드리고, 열차 몇 번째 칸에 타면 좋을지 알려드립니다. 엘리베이터가 고장나면 자동으로 다른 출구를 안내해드립니다.

---

## 어떻게 동작하나요

1. 출발역과 도착역을 입력하면 가장 편한 경로를 찾아드립니다
2. GPS로 현재 위치를 추적하면서 8단계로 안내합니다
3. 각 지점에 도착할 때마다 구체적인 안내를 받습니다
4. 엘리베이터 고장 등 실시간 상황을 반영해서 경로를 바꿔드립니다

**8단계 안내**
- 출발지 → 출발역 출구 → 개찰구 → 승강장 대기 → 열차 탑승 → 도착역 승강장 → 도착역 출구 → 목적지

---

## 기술 구성

**Backend (완료)**
- FastAPI - 경로 계산 및 안내 API
- PostgreSQL - 20개 역, 195개 출구 정보
- Seoul Open API - 실시간 엘리베이터/열차 정보 (9개 연동)
- LangChain + GPT-4 - 자연스러운 안내 메시지

**Frontend (진행중)**
- React Native - 모바일 앱
- GPS + TTS - 위치 추적 및 음성 안내
