# 길봄(Gilbom)
AI 기반 고령층 맞춤형 대중교통 네비게이션 서비스

고령층이 복잡한 교통 정보를 직접 해석하지 않아도  
단계별 안내만 따라가면 목적지까지 이동할 수 있는 새로운 형태의 길찾기 방식을 연구·개발합니다.

---

## 1. 프로젝트 개요

**길봄(Gilbom)**은  
‘길에서 도움을 주다’, ‘길을 봐준다’라는 의미를 담고 있으며,  
모든 세대가 대중교통 정보에 동등하게 접근할 수 있는 도시 환경을 만드는 것을 목표로 합니다.

서울의 풍부한 교통 인프라가 있음에도  
복잡한 정보 구조 때문에 고령층은 실제로 활용하기 어렵다는 문제를  
AI 기반 단계별 안내(step-by-step) 방식으로 해결합니다.

---

## 2. 주요 기능 소개

### 1) Step-by-Step Navigation  
- 네이버 경로 API 기반 전체 경로 수집  
- AI가 경로를 단계 단위로 분해  
- 필요한 실시간 정보(열차·버스 도착, 출구/엘리베이터 위치)를 자동 결합  
- UI는 ‘읽는 지도’가 아닌 ‘따라가는 안내’ 구조

### 2) Inclusive Mobility UI/UX  
- 고령층 시인성 기준에 맞춘 화면 구성  
- 한 화면에 하나의 안내 원칙  
- 큰 글자, 높은 대비, 불필요 정보 최소화

### 3) AI-Assisted Simplification  
- 안내 문장을 GPT가 자동 생성  
- 복잡한 교통 정보를 이해하기 쉬운 문장으로 변환  
- 지연·결번·출구 폐쇄 등 실시간 상황 반영

---

## 3. 기술 스택 & 데이터

### Core Technologies
- OpenAI GPT-4.1-mini API  
- Naver Maps Directions / Geocoding API  
- State Machine 기반 단계 전환  
- Backend: Node.js / FastAPI  
- Frontend: React  

### 활용 데이터
- 서울 열린데이터광장: 역 편의시설, 승강장, 지하철 실시간 도착  
- 공공데이터포털: 버스 정류장·실시간 도착 데이터  
- 네이버 클라우드 Maps API: 길찾기·주소 매핑

## 5. 사회·경제적 기대효과

### 1) 교통 접근성 격차 완화  
정보 이해도 차이로 인해 이동 선택지가 제한되던 고령층이  
대중교통 인프라를 정상적으로 활용할 수 있는 기반 확보.

### 2) 초고령사회 대비 AI 기반 PoC  
급속한 고령화에 대응하기 위한  
미래형 이동지원 체계의 프로토타입으로 기능하며,  
지자체 및 공공기관 확장 가능성이 큼.

## 6. Team B-FAI

B-FAI는 “Barrier-Free AI”를 의미하며  
AI를 통해 이동 정보 격차를 해소하는 것을 목표로 한 팀입니다.

### 팀 구성
- **PM** – 서비스 기획, 데이터/흐름 구조 설계  
- **Frontend Developer** – 고령친화 UI/UX 구현  
- **Backend Developer** – API 연동, 데이터 병합, 단계별 안내 시스템 개발  
- **Designer** – 정보 단순화 중심 인터페이스 설계
