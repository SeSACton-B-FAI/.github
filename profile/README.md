# 비파이 (B-FAI)

어르신과 교통약자를 위한 대중교통 안내견 서비스

---

## 왜 만들었나요

서울시의 교통 인프라는 참 편리하지만 이는 젊은 사람들에 한정된 이야기입니다. 노선수는 정말 많고 환승 방법은 너무 어렵습니다. 연구에 따르면 실제로 환승을 2회 이상 해서 도착지에 가는 경우는 경로 고객의 경우에 3% 밑에 그칩니다. 이에 비파이는 경로자분들도 서울시의 교통인프라의 혜택을 젊은 사람처럼 누릴 수 있도록 서비스를 만들었습니다.
<br>

1. 너무 많은 정보를 한번에 주지 않고 필요한 정보만 제공

2.노안으로 잘 보이지 않는데 지하철 앱이나 버스 정류장 도착 버스 안내 표지판, 지도 앱 등은 글자가 너무 작습니다.

3.최단거리가 아닌 엘레베이터가 있는 개찰구 등 어르신의 신체 연로를 고려해서 안내 드립니다.

---

## 어떻게 동작하나요

출발지는 자동으로 gps가 잡고 도착지를 입력합니다. 도착지까지의 여정을 상세히 하나하나 알려 드립니다. 

예를 들어 '강남역 2호선을 타고 성수역에서 하차하세요' 같은 정보를 '강남역 몇 번 출구로 가셔서 엘레베이터를 이용하세요.' 

'탑승장은 '성수 방면'이라고 써 있는 곳으로 내려가세요.' '약 ~분 후쯤에 성수역에 도착할 예정입니다'와 같이 풀어서 설명합니다.

**8단계 안내**
- 출발지 → 출발역 출구 → 개찰구 → 승강장 대기 → 열차 탑승 → 도착역 승강장 → 도착역 출구 → 목적지

---

## 기술 구성

**Backend (완료)**
- FastAPI - 경로 계산 및 안내 API
- PostgreSQL - 20개 역, 195개 출구 정보
- Seoul Open API - 실시간 엘리베이터/열차 정보 (9개 연동)
- LangChain + GPT-4 - 자연스러운 안내 메시지

**Frontend (진행중)**
- React Native - 모바일 앱
- GPS + TTS - 위치 추적 및 음성 안내
