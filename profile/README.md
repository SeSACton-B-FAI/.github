# 비파이 (B-FAI)

노인분들을 위한 실시간 지하철 길안내 서비스

---

## 프로젝트 소개

길 위의 노인분이 지하철을 이용할 때 어느 출구로 들어가고, 어느 방면 승강장으로 가고, 몇 번째 칸에 타야 하는지 음성으로 안내해주는 서비스입니다.

엘리베이터가 필요한 분들을 위해 엘리베이터가 있는 경로만 안내하고, 실시간으로 엘리베이터 고장 여부를 확인해서 대체 경로를 제공합니다.

---

## 주요 기능

- 계단 이용 가능 여부에 따른 최적 경로 탐색
- GPS 기반 실시간 음성 안내
- 엘리베이터 실시간 상태 확인
- 전동 휠체어 충전소 안내
- 노인 친화적 UI/UX
